安装和使用Hexo主题

本文给大家演示怎么安装和使用Hexo主题。

一、准备工作

1、在官网主题模板选一个你喜欢的主题。next是star最多的，我是选的icarus。
2、有2个配置文件，一个全局配置文件，一个是主题配置文件

二、下载主题和修改配置

修改全局配置文件（_config.yml），默认是landscape，我们换成icarus

三、生成静态文件，启动本地服务器查看

访问本机博客地址查看效果。

四、修改主题配置

将主题配置文件（themes/icarus/_config.yml）中信息换成你自己的

恭喜你的博客换成全新主题 \^o^/
